🛍️ Анализ данных интернет-магазина



Пет-проект по аналитике данных: от создания БД до дашборда в Power BI
О проекте

Цель: Провести комплексный анализ данных интернет-магазина для выявления ключевых бизнес-метрик и построения интерактивного дашборда.

Задачи:

- Разработать структуру базы данных в PostgreSQL;
- Загрузить и подготовить тестовые данные;
- Провести исследовательский анализ с помощью SQL;
- Построить дашборд в Power BI;
- Сформулировать рекомендации для бизнеса.

📊 Данные

Источник: Сгенерированные тестовые данные  

Период: Январь 2024 — Июнь 2026  

Объем данных: 530 заказов, 25 клиентов, 12 товаров
